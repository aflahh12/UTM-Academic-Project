# Probability & Statistical Data Analysis

## Course Overview
This course introduces the fundamental concepts of probability theory and statistical methods for analyzing and interpreting data. It covers descriptive statistics, probability distributions, hypothesis testing, regression analysis, and data visualization techniques essential for data-driven decision making.
