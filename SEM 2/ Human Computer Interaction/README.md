# Human-Computer Interaction (HCI)

## Course Overview
This course explores the principles of designing, evaluating, and implementing interactive systems that prioritize user needs and experiences. It covers user-centered design methodologies, usability testing, prototyping, and accessibility to create intuitive and effective digital interfaces.
