# Year 1
### [Semester 1](https://github.com/aflahh12/UTM-Academic-Project/tree/3c75e1b9610a56d80368732345f13e4e553c6cf9/SEM%201)
#### SECI1013 Discrete Structure
**Overview:** This course introduces the mathematical foundations of computer science, including logic, set theory, combinatorics, graph theory, and discrete structures essential for algorithm design and problem-solving.

**Summary of Gained Knowledge:** I developed strong logical reasoning and problem-solving skills through propositional logic, Boolean algebra, and proof techniques. I learned to apply combinatorial principles and graph theory to solve computational problems efficiently.

**Reflection:** This course strengthened my mathematical thinking and provided the theoretical foundation for many other computer science subjects. Understanding discrete structures helped me approach programming problems with a more structured and analytical mindset.

**Improvement Suggestions:** Incorporate more real-world examples, especially in graph theory, to connect abstract concepts to practical applications like social networks and routing algorithms.

---

#### SECJ1013 Programming Technique I
**Overview:** This course introduces fundamental programming concepts, problem-solving techniques, and structured programming using a high-level language, covering variables, control structures, functions, arrays, and file handling.

**Summary of Gained Knowledge:** I mastered the basics of programming including syntax, data types, decision-making, looping, and modular programming. I gained practical experience in writing structured, well-documented code to solve various computational problems.

**Reflection:** This foundational course gave me confidence in coding and thinking logically about problem-solving. Learning to break down complex problems into smaller, manageable functions was a key takeaway that I still apply today.

**Improvement Suggestions:** Introduce pair programming or small group projects to simulate real-world collaboration and improve code review skills.

**Reflection:** This course opened my eyes to the big picture of how technology integrates with business strategy. It helped me see beyond coding and understand how systems add value to organizations.

## Summary of Overall Learning Journey🔎

Throughout my three-year journey in the Bachelor of Computer Science (Data Engineering) program, I have developed a strong technical foundation combined with practical industry exposure through WBL courses. Starting from programming fundamentals and discrete mathematics in Year 1, I progressively advanced to data structures, algorithms, and software engineering in Year 2. By Year 3, I specialized in data-centric subjects including data analytics, business intelligence, high-performance data processing, and enterprise system design.

The program has shaped me into a well-rounded data engineering professional with skills in:
- Programming and software development
- Database design and programming
- Data analytics and mining
- Big data processing and distributed systems
- Business intelligence and visualization
- System analysis and enterprise architecture
