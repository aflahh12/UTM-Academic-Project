# Object Oriented Programming

## Course Overview
This course introduces the fundamental principles and concepts of object-oriented programming (OOP) using modern programming languages. It covers classes, objects, inheritance, polymorphism, encapsulation, abstraction, and design patterns to build modular, reusable, and maintainable software systems.
